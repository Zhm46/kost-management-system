Bug Report
* Title
Login gagal tidak menampilkan pesan error

* Description
Saat user memasukkan password yang salah, sistem tidak menampilkan pesan error sehingga user tidak mengetahui penyebab kegagalan login.

* Steps to Reproduce
1. Buka halaman login
2. Masukkan username yang valid
3. Masukkan password yang salah
4. Klik tombol login

* Expected Result
Sistem menampilkan pesan error bahwa password salah

* Actual Result
Sistem tidak menampilkan pesan error, form login ter-reset, dan user tetap berada di halaman login

* Severity
Medium

* Status
Open
